# 内容层级分析 Skill

基于用户提供的文档，分析并输出标准化的内容层级文档。

## 输出路径

```
docs/product-design/[项目名]/内容层级.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别信息层级规范
- 分析页面层级应用
- 检查层级一致性

### 3. 输出标准化文档
