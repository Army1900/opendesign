# 站点地图分析 Skill

基于用户提供的文档，分析并输出标准化的站点地图文档。

## 输出路径

```
docs/product-design/[项目名]/站点地图.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别所有页面/模块
- 梳理层级关系
- 分析分类逻辑

### 3. 输出标准化文档
