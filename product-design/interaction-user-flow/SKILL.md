# 用户流程分析 Skill

基于用户提供的文档，分析并输出标准化的用户流程文档。

## 输出路径

```
docs/product-design/[项目名]/用户流程.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别核心流程
- 梳理操作步骤
- 检查分支路径
- 发现流程问题

### 3. 输出标准化文档
