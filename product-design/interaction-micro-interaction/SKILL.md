# 微交互分析 Skill

基于用户提供的文档，分析并输出标准化的微交互审查文档。

## 输出路径

```
docs/product-design/[项目名]/微交互.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别交互触发点
- 分析反馈机制
- 检查动效设计
- 评估交互体验

### 3. 输出标准化文档
