# 线框图分析 Skill

基于用户提供的文档，分析并输出标准化的线框图审查文档。

## 输出路径

```
docs/product-design/[项目名]/线框图分析.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别页面布局结构
- 分析信息层级
- 检查元素可见性
- 评估交互区域

### 3. 输出标准化文档
