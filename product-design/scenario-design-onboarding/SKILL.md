# 新手引导场景分析 Skill

基于用户提供的文档，分析并输出标准化的新手引导场景文档。

## 输出路径

```
docs/product-design/[项目名]/新手引导.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别首次使用的关键节点
- 分析学习成本和认知负担
- 梳理引导流程和方式

### 3. 输出标准化文档
