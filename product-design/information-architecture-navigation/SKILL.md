# 导航结构分析 Skill

基于用户提供的文档，分析并输出标准化的导航结构文档。

## 输出路径

```
docs/product-design/[项目名]/导航结构.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别导航类型
- 检查导航一致性
- 评估导航可用性

### 3. 输出标准化文档
