# 状态设计分析 Skill

基于用户提供的文档，分析并输出标准化的状态设计审查文档。

## 输出路径

```
docs/product-design/[项目名]/状态设计.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别组件类型
- 检查状态覆盖
- 评估状态一致性
- 发现遗漏状态

### 3. 输出标准化文档
