# 提示反馈分析 Skill

基于用户提供的文档，分析并输出标准化的提示反馈审查文档。

## 输出路径

```
docs/product-design/[项目名]/提示反馈.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别提示类型
- 分析提示触发条件
- 检查文案质量
- 评估用户体验

### 3. 输出标准化文档

---
