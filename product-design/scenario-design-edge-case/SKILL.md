# 边缘场景分析 Skill

基于用户提供的文档，分析并输出标准化的边缘场景文档。

## 输出路径

```
docs/product-design/[项目名]/边缘场景.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别可能的异常情况
- 检查空状态设计
- 分析错误处理机制
- 评估边界条件

### 3. 输出标准化文档
