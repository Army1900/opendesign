# 上下文场景分析 Skill

基于用户提供的文档，分析并输出标准化的上下文场景文档。

## 输出路径

```
docs/product-design/[项目名]/上下文场景.md
```

## 工作流程

### 1. 接收输入
- HTML 文件夹 / 压缩包 / 文档

### 2. 分析内容
- 识别不同使用环境
- 分析时间和空间维度
- 检查设备和网络适配

### 3. 输出标准化文档

---
