# 微交互分析模块

分析产品的微交互设计，梳理交互细节和反馈。

**每个章节必须包含两部分**：
1. **当前设计梳理**：从设计稿中提取的现有微交互设计
2. **审查意见**：基于梳理内容给出的问题、风险、建议

## 检查项清单

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 点击反馈 | 所有可点击元素有反馈 | 高 | 点击无反应 |
| 加载状态 | 耗时操作有加载提示 | 高 | 长时间无反馈 |
| 状态切换 | 切换动画流畅 | 中 | 生硬切换 |
| 手势支持 | 关键手势有支持 | 中 | 无手势 |
| 手势引导 | 新手势有引导 | 中 | 不知道有手势 |
| 动效时长 | 时长在合理范围 | 低 | 过长或过短 |
| 触觉反馈 | 重要操作有震动 | 低 | 无触觉反馈 |
