# 安全设计分析

分析产品的安全设计，梳理隐私保护、数据安全、权限控制和合规要求。

---

## 输出结构

**每个章节必须包含两部分**：
1. **当前设计梳理**：从PRD/设计文档中提取的现有安全设计
2. **审查意见**：基于梳理内容给出的问题、风险、建议

## 检查项清单

### 认证安全检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 多因素认证 | 敏感操作需要二次验证 | 高 | 无二次验证 |
| 密码强度 | 密码复杂度要求 | 高 | 弱密码可注册 |
| 密码存储 | 加盐哈希，不明文存储 | 高 | 明文存储密码 |
| 登录保护 | 错误锁定、验证码保护 | 高 | 无防暴力破解 |
| 会话管理 | Token过期、刷新机制 | 高 | Token永不过期 |
| 登出完整 | 服务端Session失效 | 中 | 仅前端清缓存 |

### 数据安全检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 敏感数据加密 | 数据库加密存储 | 高 | 敏感数据明文存储 |
| 数据脱敏 | 显示时脱敏处理 | 高 | 敏感信息直接显示 |
| 数据分类 | 按敏感级别分类 | 中 | 无分类管理 |
| 数据最小化 | 只采集必要数据 | 中 | 过度采集 |
| 数据保留 | 定义保留期限 | 中 | 数据永久保留 |
| 数据销毁 | 账号注销时数据删除 | 中 | 注销后数据不清除 |

### 权限控制检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 最小权限原则 | 只授予必要权限 | 高 | 权限过大 |
| 水平越权 | 同角色用户数据隔离 | 高 | 可查看他人数据 |
| 垂直越权 | 不同角色权限隔离 | 高 | 低权限可执行高权限操作 |
| 权限校验 | 每次操作都校验 | 高 | 仅前端隐藏 |
| 敏感操作二次验证 | 敏感操作需确认 | 高 | 无二次验证 |

### 接口安全检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| HTTPS | 全站HTTPS | 高 | 部分接口HTTP |
| 参数校验 | 入参格式/范围校验 | 高 | 无校验直接使用 |
| SQL注入防护 | 参数化查询 | 高 | 字符串拼接SQL |
| XSS防护 | 输入过滤输出转义 | 高 | 直接输出用户输入 |
| CSRF防护 | Token校验 | 高 | 无CSRF防护 |
| 接口限流 | 防止恶意请求 | 中 | 无限流保护 |
| 接口签名 | 敏感接口签名验证 | 中 | 无签名校验 |

### 隐私合规检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 隐私政策 | 有完整隐私政策 | 高 | 无隐私政策 |
| 用户授权 | 采集前获得授权 | 高 | 未授权采集 |
| 用户知情 | 告知采集目的和范围 | 高 | 未告知 |
| 访问权 | 用户可查看自己的数据 | 中 | 无查看入口 |
| 删除权 | 用户可删除自己的数据 | 中 | 无删除功能 |
| 撤回同意 | 用户可撤回授权 | 中 | 无法撤回 |
| 数据导出 | 用户可导出自己的数据 | 低 | 无导出功能 |

### 日志审计检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 登录日志 | 记录登录行为 | 中 | 无登录日志 |
| 敏感操作日志 | 记录敏感操作 | 中 | 无操作日志 |
| 日志完整性 | 日志不可篡改 | 中 | 日志可删除 |
| 日志保留 | 保留足够时间 | 中 | 保留时间不足 |
| 异常告警 | 异常行为告警 | 低 | 无告警机制 |

## 执行要点

1. 读取 PRD/安全规范/隐私政策
2. 识别敏感数据类型和级别
3. 检查认证和密码安全机制
4. 检查数据脱敏和加密措施
5. 检查权限控制设计
6. 检查接口安全防护
7. 检查合规要求满足情况
8. **对照检查项清单逐项审查**
9. 输出带状态和风险的审查结果
10. 标注高风险项和待确认项
