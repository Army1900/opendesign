## 检查项清单

### 实体设计检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 聚合边界 | 聚合根和内部实体边界清晰 | 中 | 聚合过大或过小 |
| ID 生成策略 | UUID/雪花算法/自增 | 中 | 分布式 ID 冲突 |
| 乐观锁 | 版本号/时间戳并发控制 | 高 | 并发更新覆盖 |
| 软删除 | 删除标记而非物理删除 | 低 | 数据无法恢复 |
| 审计字段 | 创建人/创建时间/更新人/更新时间 | 低 | 无审计追踪 |
| 数据校验 | 字段格式/范围/唯一性校验 | 中 | 无校验或校验不完整 |
| 枚举管理 | 枚举值集中管理 | 低 | 魔法数字 |
| 值对象 | 复用值对象封装 | 低 | 重复定义 |

### 存储设计检查项

| 检查项 | 说明 | 风险等级 | 常见问题 |
|--------|------|----------|----------|
| 存储选型 | 关系/文档/时序/图数据库 | 高 | 选型不当 |
| 表设计规范 | 命名规范、字段类型合理 | 中 | 命名混乱 |
| 主键策略 | 自增/UUID/雪花 | 中 | 分布式冲突 |
| 索引策略 | 查询覆盖、避免全表扫描 | 高 | 无索引或索引过多 |
| 分库分表 | 数据量大时的分片策略 | 高 | 单表过大 |
| 读写分离 | 读多写少场景 | 中 | 主库压力过大 |
| 连接池配置 | 最大连接数、超时时间 | 高 | 连接泄露 |
| 事务隔离级别 | RC/RR/Serializable | 高 | 脏读/幻读 |
| 备份策略 | 全量+增量备份 | 高 | 无备份 |
| 容量预估 | 数据增长预测 | 中 | 空间不足 |
