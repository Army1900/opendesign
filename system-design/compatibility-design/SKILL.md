## 兼容性维度

### 1. 浏览器兼容性
| 维度 | 检查项 | 常见问题 |
|------|--------|----------|
| 内核 | Trident/EdgeHTML | IE11 语法不支持 |
| 引擎 | V8/SpiderMonkey | ES6+ 语法差异 |
| 版本 | Chrome版本 | 新 API 旧版不支持 |
| 移动端 | iOS Safari / Android Chrome | WebView 内核差异 |

### 2. 设备兼容性
| 维度 | 检查项 | 常见问题 |
|------|--------|----------|
| 屏幕尺寸 | 4.7" ~ 12.9" | 布局适配 |
| 分辨率 | 1x ~ 4x | 图片模糊 |
| 内存 | 2GB ~ 16GB | 大图片/数据崩溃 |
| CPU | 单核 ~ 多核 | 性能差异 |

### 3. 系统兼容性
| 维度 | 检查项 | 常见问题 |
|------|--------|----------|
| iOS | 12 ~ 17 | API 差异、 安全策略 |
| Android | 8 ~ 14 | 权限模型、 通知机制 |
| Windows | 7 ~ 11 | 系统调用差异 |
| macOS | 10 ~ 14 | 文件系统差异 |

### 4. 数据兼容性
| 维度 | 检查项 | 常见问题 |
|------|--------|----------|
| API版本 | v1 / v2 | 字段变更、 废弃接口 |
| 数据格式 | JSON / XML | 结构差异 |
| 编码 | UTF-8 / GBK | 乱码问题 |
| 时间格式 | 时间戳 / ISO8601 | 时区问题 |

### 5. 服务兼容性
| 维度 | 检查项 | 常见问题 |
|------|--------|----------|
| 第三方API | 版本升级 | 接口变更 |
| 支付渠道 | 支付宝/微信 | 接入差异 |
| 地图服务 | 高德/百度 | SDK 差异 |
| IM服务 | 极光/融云 | 推送机制 |

### 6. 网络兼容性
| 维度 | 检查项 | 常见问题 |
|------|--------|----------|
| 协议 | HTTP/1.1 / HTTP/2 | 头部字段差异 |
| 安全 | TLS 1.2 / 1.3 | 加密套件差异 |
| 代理 | 正向/反向代理 | 请求头处理 |
| 缓存 | 强缓存/协商缓存 | 缓存策略差异 |

## 兼容性问题识别规则

### 浏览器问题
```javascript
// 识别 ES6+ 语法使用
const features = ['async/await', 'Promise', '箭头函数', '解构赋值', 'ES Modules'];

// 识别 CSS 新特性
const cssFeatures = ['CSS Grid', 'CSS Variables', 'flexbox', 'gap'];
```

### 设备问题
```javascript
// 识别设备特性调用
const deviceAPIs = ['geolocation', 'camera', 'bluetooth', 'nfc', 'biometric'];

// 识别屏幕适配
const responsivePatterns = ['media query', 'viewport', 'dpr'];
```

### API 问题
```java
// 识别版本化 API
String[] apiPatterns = {"/v1/", "/v2/", "/api/v1/", "version="};
```
